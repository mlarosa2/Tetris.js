# Tetris.js

A browser Tetris game implemented with vanilla JavaScript and the HTML5 Canvas API.

[Play Now!](http://www.marklarosa.me/tetris.html)

## Controls:

  * "a" move block left
  * "s" move block down
  * "d" move block right
  * "q" rotate counter clockwise
  * "e" rotate clockwise
  * "p" pauses game
